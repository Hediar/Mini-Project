## 실행방법
app.py로 flask 준비 → 가상환경 준비(venv) python -m venv venv → templates폴더에 index.html 준비 → 각종 패키지 설치 pip install ~~ → 큰 틀 준비 완료

vscode의 경우 f5로 디버깅 실행

localhost:5000 접속
